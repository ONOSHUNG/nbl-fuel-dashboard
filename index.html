วิธีเชื่อม Google Sheet (ทำครั้งเดียว)
1
เปิด Google Sheet ที่ต้องการ → เมนู Extensions → Apps Script
2
ลบ code เดิมทั้งหมด แล้ว วาง code จากไฟล์ Code.gs ที่ได้รับ
3
กด Deploy → New deployment → เลือก Type: Web app → Execute as: Me → Who has access: Anyone → กด Deploy
4
คัดลอก Web app URL ที่ได้ (ขึ้นต้นด้วย https://script.google.com/macros/s/...)
5
วาง URL ในช่องด้านล่าง แล้วกด "ทดสอบและบันทึก"
